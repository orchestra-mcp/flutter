# Internationalization (i18n)

EN + AR locales. Locale persisted via SharedPreferences, reactive via Riverpod.

## Usage

```dart
// Watch current locale
final locale = ref.watch(localeProvider);

// Change locale
await ref.read(localeProvider.notifier).setLocale(const Locale('ar'));

// RTL helpers
RtlUtils.isRtl(context);          // bool
RtlUtils.textAlign(context);      // TextAlign.right or .left
RtlUtils.dirIcon(context, ltr: Icons.arrow_forward, rtl: Icons.arrow_back);
RtlUtils.symmetric(context: context, start: 16, end: 8); // EdgeInsets
```

## Supported Locales

| Code | Language |
|------|----------|
| en   | English  |
| ar   | Arabic (RTL) |

## ARB Keys

`lib/l10n/app_en.arb` is the template (263 keys, 19 namespaces). Arabic translations in `lib/l10n/app_ar.arb`.

| Namespace | Key examples |
|-----------|-------------|
| common | ok, cancel, save, delete, search |
| auth | login, register, forgotPassword, magicLogin, passkey |
| settings | appearance, language, theme, signOut, mcpPort |
| projects | newProject, features, plans, projectTree |
| health | hydration, caffeine, pomodoro, shutdown, vitals |
| errors | networkError, serverError, authError, syncError |

## l10n.yaml

```yaml
arb-dir: lib/l10n
template-arb-file: app_en.arb
output-localization-file: app_localizations.dart
```

Generated class: `AppLocalizations` (access via `AppLocalizations.of(context)!`).
